# INFNOVA Academy - Responsive Courses Website

A clean, responsive static website for an online learning platform, built with HTML5 and CSS3 only (no frameworks).

## Features

- Fully responsive design (desktop, tablet, mobile)
- Course listing page with searchable grid layout
- Detailed course view with hero section, sidebar enroll box, and right-aligned call-to-action
- Pixel-perfect match to provided UI mockups
- Modern CSS (Flexbox + Grid)
- Semantic HTML structure

## Pages

- `index.html` → Courses explorer / listing page  
- `course.html` → Single course detail page (example: Cloud Engineering with AWS)

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- No JavaScript / No external libraries
